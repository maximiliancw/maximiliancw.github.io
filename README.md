# Personal Website

This repository contains the source code for my personal website, hosted at [maximiliancw.github.io](https://maximiliancw.github.io).

## About

A lightweight, single-page website showcasing my professional background, experience, and expertise as an IT-Business Translator & Innovation Leader.

## Technology

- Pure HTML, CSS, and JavaScript (no build tools or frameworks)
- GitHub Pages for hosting
- GitHub API integration for open-source contributions display

## Structure

The website includes sections for:

- Professional highlights
- Career timeline
- Education
- Skills & tools
- Open-source contributions
- Freelancing & consulting services
- Gallery of impressions
- Contact information
